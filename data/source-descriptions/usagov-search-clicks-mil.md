## USA.gov Search Clicks - .mil

Search.gov provides a gov-wide search engine for usa.gov and is able to generate a snapshot of the pages that are found and clicked on through this. This is a snapshot of the .mil domains in that dataset from February 2024.  

[The file is located here](https://github.com/GSA/federal-website-index/blob/main/data/source-lists/usagov-search-clicks-mil.csv).
